Enterprise Email Security Implementation

Automated quarantine digest, threat intel enrichment, and live KPIs for Microsoft 365.

Arc Innovations · 2025 · Power Automate · Microsoft Graph · Security · Analytics

TL;DR: We built a secure pipeline that ingests quarantine items, enriches with sender/IP intel, and publishes a daily Quarantine Digest plus a lightweight dashboard for trends and approvals.

Overview

The solution fetches email threat data (spam, phishing, malware) from Microsoft Defender for Office 365 / Graph, applies enrichment and allow/deny rules, and generates a digest for end users and admins. Actions are logged with correlation IDs for audit.
